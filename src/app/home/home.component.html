Income statement report service

To start:
You can use income statement report service with or without an account

Without an account:

  • Your data is not save into the databases and it will be gone after closing the browser
  • All uploaded transactions will be included in the report and report period cannot be definer
  • Rule functionality to speed up the transaction type defining process is disabled

How to use income statement report service without registering new account:

  • Download your bank account statement file in CVS or XML format from the internet bank
  • Upload your file here
  • Define income statement types for your transactions
  • Get the income statement report which values you can insert to Äriregister

Welcome back!

How to use income statement report service with an account:

  • Download your bank account statement file in CVS or XML format from the internet bank
  • Upload your file here
  • Define income statement types for your transactions
  • Define rules to speed up the transaction type defining process
  • Get the income statement report for your defined period
  • Insert report values to Äriregister
