How to use income statement service

With an account:
  • Download your bank account statement file in CSV or XML format from the internet bank
  • Upload your file here
  • Define income statement types for your transactions
  • Define rules to speed up the transaction type defining process
  • Get the income statement for your defined period
  • Insert report values to Äriregister
Without an account:
  • Download your bank account statement file in CSV or XML format from the internet bank
  • Upload your file here
  • Define income statement types for your transactions
  • Get the income statement which values you can insert to Äriregister

What is the difference? Without an account:

  • Your data is not save into the database and it will be deleted after closing the browser
  • All uploaded transactions will be included in the report and report period cannot be defined
  • Rule functionality to speed up the transaction type defining process is disabled
